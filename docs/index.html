SIEM on Amazon OpenSearch Service

A solution for collecting multiple types of logs from multiple AWS accounts, correlating and visualizing them to help investigate security incidents.

日本語 简体中文 繁體中文

SIEM on Amazon OpenSearch Service is a solution for collecting multiple types of logs from multiple AWS accounts, correlating and visualizing the logs to help investigate security incidents. Deployment is easily done with the help of AWS CloudFormation or AWS Cloud Development Kit (AWS CDK), taking only about 30 minutes to complete.
